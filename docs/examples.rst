Examples
========

Here are some examples of bots using *guirecognizer*. The code for each example is available at
`https://github.com/fab5code/guirecognizer/tree/main/examples <https://github.com/fab5code/guirecognizer/tree/main/examples>`_.
Is described the processus to retrieve information from the screen using *guirecognizerapp* and *guirecognizer*.
The algorithms to build the rest of the bot are not explained.

- :doc:`example/otteretto`: A game about palindromes on a grid.
